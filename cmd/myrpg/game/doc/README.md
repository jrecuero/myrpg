## Quick Reference

### Action Points by Job Class
| Job | Max AP | Attacks/Turn |
|-----|--------|--------------|
| Warrior | 4 | 2 |
| Mage | 3 | 1 |
| Rogue | 5 | 2 |
| Cleric | 4 | 2 |
| Archer | 4 | 2 |

### Action Costs
- **Attack**: 2 AP
- **Move**: 1 AP per tile
- **Item Use**: 1 AP
- **End Turn**: Free (0 AP)

### Key Controls
- **A**: Attack (Tactical mode)
- **E**: End Turn (Tactical mode)
- **T**: Toggle Tactical Mode
- **I**: Toggle Inventory
- **K**: Toggle Skills Window
- **Arrow Keys**: Movement
- **Space**: Interact with NPCs
